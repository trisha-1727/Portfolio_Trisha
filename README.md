# Portfolio_Trisha
# Personal Portfolio Website

## Overview
This project is a personal portfolio website built using **HTML and CSS**, designed to showcase my skills, education, and projects in a structured and accessible format. It serves as a digital resume and provides an easy way for recruiters and collaborators to learn more about me and get in touch.

🔗 **Live Website:**
[https://trisha-1727.github.io/Portfolio_Trisha/](https://trisha-1727.github.io/Portfolio_Trisha/)

---

## Project Structure
* `index.html` – Main portfolio page with summary, education, skills, and projects 
* `about.html` – Detailed introduction and background information 
* `contact.html` – Contact details including email and LinkedIn 
* `styles.css` – Styling and layout of the website 


## Tools & Technologies
* **HTML5** – Structure and content
* **CSS3** – Styling and layout design
* **GitHub Pages** – Hosting and deployment


## Features
* Clean and simple user interface
* Multi-page navigation (Home, About, Contact)
* Responsive layout basics using viewport settings
* Structured presentation of resume details (education, skills, projects)
* Direct links to GitHub and LinkedIn for easy access


## Purpose & Importance

* Demonstrates foundational skills in **frontend web development**
* Acts as a **personal branding tool** for recruiters and networking
* Showcases ability to build and deploy a live project using GitHub
* Provides a centralized platform for projects and professional information
